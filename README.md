### [`pd_sparse_multiclass`](http://www.cs.utexas.edu/~xrhuang/PDSparse/):
- aloi.bin 
- Dmoz 
- imageNet 
- LSHTC1 
- sector

### [`pd_sparse_multilabel`](http://www.cs.utexas.edu/~xrhuang/PDSparse/):
- Eur-Lex 
- rcv1_regions 
- bibtex
- LSHTCwiki

### [`xml_repo`](https://manikvarma.github.io/downloads/XC/XMLRepository.html):
- Amazon-3M 
- AmazonCat 
- Bibtex 
- DeliciousLarge
- Delicious
- Mediamill
- RCV1-x
- Wiki10
- WikiLSHTC
